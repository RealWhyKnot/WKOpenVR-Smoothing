# OpenVR-WKSmoothing

Source has moved to [OpenVR-WKPairDriver](https://github.com/RealWhyKnot/OpenVR-WKPairDriver), the umbrella repository for the OpenVR-Pair toolset.

This repo is kept as a release mirror. Each release here is the OpenVR-Pair umbrella build with `enable_smoothing.flag` pre-dropped so finger smoothing and pose prediction activate immediately on install.

Pre-migration history is preserved on the [`archive-source`](https://github.com/RealWhyKnot/OpenVR-WKSmoothing/tree/archive-source) branch and the existing tags.

Issues and contributions: open them at [OpenVR-WKPairDriver](https://github.com/RealWhyKnot/OpenVR-WKPairDriver/issues).
