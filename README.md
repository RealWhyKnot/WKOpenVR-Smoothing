# WKOpenVR Smoothing

Smoothing is the WKOpenVR module for reducing noisy finger input and suppressing per-device pose prediction where it causes visible tracker motion.

Download the latest `WKOpenVR-Smoothing-v...-Setup.exe` from Releases and run it. The installer installs WKOpenVR, registers the SteamVR driver, and enables Smoothing. Other WKOpenVR modules can be enabled from the Modules tab or by running another module installer.
