# WKOpenVR-Smoothing

Source has moved to [WKOpenVR](https://github.com/RealWhyKnot/WKOpenVR), the umbrella repository for the WKOpenVR feature modules.

This repo is kept as a release mirror. Each release here ships both a `WKOpenVR-Smoothing-<version>.zip` (the WKOpenVR umbrella build with `enable_smoothing.flag` pre-dropped so finger smoothing and pose prediction activate immediately on install) and a `WKOpenVR-Smoothing-v<version>-Setup.exe` (NSIS installer that does the same plus a Start Menu shortcut and an uninstaller).

This module has no upstream -- the smoothing code was authored in [WKOpenVR](https://github.com/RealWhyKnot/WKOpenVR). Pre-monorepo source state is preserved in the umbrella repo's git history.
